// HOW THIS IS CALCULATED

Your expected payout = (your hashrate ÷ network hashrate) × network PRL minted in the window × luck × (1 − pool fee). Network emission follows Pearl's continuous curve — there are no halving cliffs — so the reward minted between block heights a and b is cumulative(b) − cumulative(a), where cumulative(h) = S·h ⁄ (h + C), with S = 2.1B PRL total supply and C ≈ 650,226 (the half-supply block). This reproduces the chain's current block reward to within a fraction of a percent.

Sources: live network hashrate, block height, block reward and PRL price are fetched on-chain from PearlTrack (/api/summary). The emission curve and the ≈650,226 half-supply block are documented by PearlTrack and Lord of Pearls; Pearl's 2.1B max supply, declining reward and ~2-minute (194s target) blocks are described by Hashrate Index. Hashrate is measured as matmul attempts per second. Per-GPU rates are community estimates and vary with tuning. Pool fee is read live from the pool. Aggregator prices for "PRL" can reference the wrong token, so we use the Pearl-native explorer price.
